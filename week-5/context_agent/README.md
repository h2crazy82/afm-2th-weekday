# [Context+Agent+DB] 나를 아는 AI 에이전트

AI Context(.md)에 내 프로필 + DB 데이터 결합 → 맞춤형 답변하는 에이전트.
Context **없이** vs **있이** 비교 시연 포함.

- 원본: https://www.notion.so/c6a362eb013482b2b0cc012d2b2cbdc5
- 시나리오 예: 내 식단 아는 영양사 / 내 취업 코치 / 내 독서 큐레이터

실행: `cp .env.example .env` → 키 채우기 → `node --env-file=.env server.js` → http://localhost:3000
