# 프롬프트 — [Context+Agent+DB] 나를 아는 AI 에이전트

> 아래 내용을 Claude Code에 그대로 붙여넣고 `{{...}}` 를 본인 값으로 채워서 실행.

---

## 전제
- 수업 튜토리얼 "내 AI Context 만들기", "대시보드(.md) + 세부데이터 구조" 완료
- Supabase 프로젝트 준비

## Part 1 — AI Context 파일 작성
`context.md` 를 만들고 아래 구조로 내 프로필 입력.

```markdown
# {{내 이름}} 의 프로필

## 기본
- 직업: {{예: 라라케어 대표 / AI 부트캠프 수강생}}
- 관심사: {{예: AI 자동화, 교육, 사업운영}}
- 현재 목표: {{예: 3개월 안에 월매출 N 달성 / 부트캠프 25pt 풀클리어}}

## 루틴
- {{예: 아침 9시 뚜레쥬르 청구 / 저녁 8시 타자 연습}}

## 제약사항
- {{예: 유당불내증 / 예산 월 30만원 / 저녁 7시 이후 회의 불가}}

## 선호
- {{예: 간결한 답변 / 마크다운 표 / 한국어}}
```

## Part 2 — DB 데이터
주제 선택: **{{가계부 / 식단 / 학습 기록 / 운동 / 독서 — 택 1}}**

- Supabase에 관련 테이블 생성 (10건 이상 데이터)
- 예) 식단: `meals` (date, menu, kcal, category)

## Part 3 — 에이전트 구성
에이전트가 `context.md` + DB 데이터 **둘 다** 읽어서 답변하도록 구성.
시스템 프롬프트 가이드: "사용자의 `context.md` 를 먼저 읽고, 질문에 답할 때 DB를 조회해서 개인화된 답변을 하라."

## Part 4 — Before / After 비교
아래 2개 이상 질문을 **(a) Context 없이 일반 AI** vs **(b) 내 에이전트** 양쪽에 물어보고 답변 캡처.

{{본인 주제에 맞게 질문 채우기}}

**질문 예시 (식단 주제):**
1. "오늘 뭐 먹을까?"
2. "이번 주 단백질 섭취 괜찮아?"

**질문 예시 (가계부 주제):**
1. "이번 주말 쇼핑 얼마까지 써도 돼?"
2. "다음 달 여행 예산 어떻게 짜면 좋을까?"

**질문 예시 (학습 주제):**
1. "다음 주 뭐 공부하면 좋을까?"
2. "내가 부족한 부분 뭐야?"
