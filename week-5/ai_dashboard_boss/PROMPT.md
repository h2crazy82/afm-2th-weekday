## 기술 스택
- Backend: Express + Supabase + Notion SDK(선택) + OpenAI
- Frontend: `public/dashboard.html` (로그인 필수)
- 배포: Vercel

## Part 2 — 데이터 소스 2개 이상 연결
아래에서 2개 선택 (3개면 만점에 가까움):

- [ ] **Notion MCP / Notion API** — 내 할일/메모 (`week-5/mcp_notion_ai/` 패턴)
- [ ] **Supabase DB** — 가계부 / 습관 / 메모 (`week-5/account_book/` 재활용)
- [ ] **외부 API** — 날씨(OpenWeather) / 뉴스(NewsAPI) / 환율 등

**내 선택:**

## Part 3 — AI 브리핑
연결된 데이터를 종합해서 "오늘의 브리핑" 섹션 생성. 예:
> "좋은 아침! 오늘 할일 5개 (기획서 마감 임박), 이번 주 지출 예산 80% 도달. 점심은 가볍게."

OpenAI API 사용. 프롬프트 가이드: "다음 데이터를 보고 사용자에게 2~3문장으로 브리핑해라."
