# Screenshots

Chrome MCP가 캡처한 4장의 스크린샷 (Docebo / Hunet / Sana / AI 대화)은 Claude Code 세션 내 인라인 이미지로 캡처되었습니다.

## 디스크 파일로 옮기는 방법

### 옵션 A — 브라우저에서 cmd+shift+4
3개 사이트를 다시 방문해서 각각 cmd+shift+4로 영역 캡처:
1. https://www.docebo.com/ → `01-docebo-landing.png`
2. https://www.hunet.co.kr/ → `02-hunet-landing.png`
3. https://sanalabs.com/ → `03-sana-landing.png`
4. Claude Code 터미널 화면 → `04-ai-research-chat.png`

### 옵션 B — Screen Recording 권한 부여 후 자동
시스템 설정 → 개인정보 보호 → 화면 기록 → Terminal/iTerm/Cursor에 권한 부여 후
`screencapture` 명령으로 자동 저장 가능.
