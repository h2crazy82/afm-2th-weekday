## 1. 휴넷의 5개 핵심 자산

휴넷이 "NO.1 L&D Company"를 자임하는 근거가 되는 5개 자산을 기능 단위로 분해.

### 1.1 멀티 사이트 구조 (Domain Split)

```
hunet.co.kr           — B2C + 통합 브랜드 허브 (학점은행/공개교육/CEO/MBA 등)
hrd.hunet.co.kr       — B2B 전용 (휴넷기업교육) — 기업 결재자 진입 ★
```

→ B2B/B2C 영업 메시지를 도메인 차원에서 분리. SEO + 결재자 워크플로우 최적화.

### 1.2 카테고리 트리 (3단계 분류)

#### hunet.co.kr 메인 — 7개 라인 + 1개 멤버십

| 구분 | 라인 | 비고 |
|------|------|------|
| 기업교육 | hrd.hunet.co.kr 별도 사이트 | B2B 전용 |
| 휴넷CEO | 임원·리더 토탈 솔루션 | 구독형 씽크탱크 (1개월 무료) |
| 휴넷MBA | Executive MBA + 임원 리더십 강화 | 학위형 |
| 하이브리드러닝 | 온오프 결합 | 콘텐츠 형식 분류 |

### 1.3 스마트견적 — Self-Serve 견적 시스템

**플로우**:
```
필터 선택 (대/중/소분류 + 난이도 + 환급/비환급)
  ↓
과정 검색 → 표 형태 결과 (페이지네이션 1~10+)
  ↓
다중 체크박스 → "선택과정 견적확인" 또는 "검색결과 다운로드"
```

**과정 표 컬럼**:
| 컬럼 | 의미 |
|------|------|
| 대/중/소분류 | 3단계 카테고리 |
| 과정명 | "[북러닝] 리더십 트리거(도서 제공)교재" 같이 형식+가치 표시 |
| 학습기간 | 1개월 등 |
| 학습시간 | 3시간 / 8시간 / 40시간 등 |
| 교육비 | 60,000~250,000원 (단과) / 1,250,000원 (스쿨형) |
| **대기업환급금** | 환급 금액 노출 |
| **우선지원환급금** | 중소기업/중견기업 가산 환급금 노출 |

**필터 옵션**:
- 난이도: 기본 / 일반 / 심화
- 환급/비환급: 고용보험 환급과정 / 비환급과정

**한계 명시**:
> "스마트 견적은 정가 기준 예상 견적입니다. 교육 인원이나 니즈에 따라 맞춤 상담해 드립니다.
> 법정의무교육 과정 견적은 법정교육 다이렉트 페이지에서 확인하세요."

→ **인원수별 협상은 1:1 상담**. 정가 산출은 즉시, 협상은 사람.

### 1.4 가격 패턴

| 형태 | 가격대 | 시간 | 비고 |
|------|--------|------|------|
| 일반 단과 | 60,000~120,000원 | 2-5시간 / 1개월 | 가장 큰 비중 |
| 북러닝 (도서 제공) | 70,000~130,000원 | 3-4시간 + 도서 | 콘텐츠 + 굿즈 결합 |
| hunet MBA 단과 | 250,000원 | 8시간 | 5개 모듈 |
| hunet MBA 종합 | 1,250,000원 | 40시간 (5개월) | 단과 5개 = 1,250,000 (정확히 5배, 묶음 할인 없음 명시) |

→ 단과 진입가 6만원, 학위형까지 1,250만원. 환급 후 실부담은 절반 이하.

### 1.5 휴넷 FLEX 멤버십 (서브스크립션)

```
법정교육 무료 + 직무교육 무제한
중견/중소기업 전용
```

→ 가격 비공개 (HRD 담당자가 견적 요청). 단발 수강료 vs 멤버십 어느 쪽이 유리한지 자동 비교 도구는 없어 보임.

### 1.6 터닝포인트 큐브 — 진단 도구 (NEW)

```
"리더를 평가하지 않습니다. 리더십을 진단합니다."
Learning + Assessment + Development 3축 모델
```

→ **개인 리더 단위 진단**. 회사 전체 컴플라이언스 진단은 아님. 진단 → 교육 → 개발 풀 사이클.

## 2. 콘텐츠 트렌드 (2026년 4월 기준)

### 2.1 AI 콘텐츠 폭발

스마트견적에서 발견된 AI 관련 신규 콘텐츠:

```
- GPT만으론 부족하다면? 제미나이 실전 활용 전략
- AI 활용을 위한 최소한의 컴퓨터 지식 쌓기
- 마케터를 구한 AI 친구들: 나만의 마케팅 팀 만들기
- [일머리스쿨] AI로 완성하는 일잘러의 보고서
- 글 잘 쓰는 AI Agent 만들기 : 비즈니스 문서 편
- 윤자동과 함께하는 MAKE 업무 자동화
- AI로 완성하는 프로젝트 관리 스킬
- [투자의 신] AI로 부자되는 ETF 투자 전략
- 바이브코딩으로 시작하는 AI 웹크롤링
```

→ **월 5-10개 신규**. 콘텐츠 추가 속도가 빠름.

휴넷 AI 칼리지 별도 카테고리 운영:
- 데이터 시각화 (엑셀 + ChatGPT)
- AI 활용 교육 (GPT-4o 이미지 생성)
- 요즘 마케터 ChatGPT 활용
- AI 부업 / AI 동물 숏폼
- AI 워크플로우 (Make + ChatGPT)
- AI 에이전트 업무자동화

### 2.2 리더십 시리즈 (스테디셀러)

```
- 리더십저니
- 터닝포인트 팀장리더십스쿨 (★ 스테디셀러)
- 행복한북클럽 통찰 vol.2
- 휴넷 촌철활인 (220만 독자, 20주년 특별개정판)
```

→ B2B 영업의 강력한 무기. 책 + 강의 묶음.

### 2.3 언론 보도 (2026년 1Q)

```
2026-03-22 휴넷, AI 기반 사례 분석·토론 기능 도입 — 맞춤형 MBA 학습 강화
2026-03-17 중소기업 인력난 속 전문가 인재 매칭 플랫폼 '탤런트뱅크' 눈길
2026-03-16 휴넷, 리더십 진단 솔루션 '터닝포인트 큐브' 출시 — 리더십 리스크까지 진단
```

→ 휴넷 전략 방향: **AI 기반 사례 분석 + 매칭 플랫폼 + 진단 솔루션**. 콘텐츠 마켓을 넘어 도구 SaaS 영역으로 확장 중.

## 6. 핵심 인사이트 — "휴넷이 가는 길" + 라라에듀 함의

### 6.1 휴넷의 전략 진화 (2026년 1Q 기준)

```
콘텐츠 마켓 (2010~) →
멤버십 (휴넷 FLEX, 2020~) →
진단 도구 (터닝포인트 큐브, 2026 신규) →
AI 사례 분석·토론 (맞춤형 MBA, 2026 신규)
```

→ **콘텐츠 매출 → 도구·SaaS 매출**로 확장 중. 라라에듀가 7주차 MVP에서 노리는 영역이 이미 휴넷의 차기 영역.

### 6.2 함의 — 시간이 핵심

휴넷이 6-12개월 안에 "회사 단위 컴플라이언스 진단 + AI 콘텐츠 자동 맞춤화"로 확장할 가능성 높음.

라라에듀가 가져갈 6-12개월 윈도우:
- **1번 강점은 한국 제조업 데이터** (252사 + 산업안전 사고)
- **2번 강점은 견적 자동 발송 + 단가 로직** (휴넷은 1:1 상담 의존)
- **3번 강점은 라라에듀가 작은 조직** (의사결정 속도, 휴넷은 대형 조직)

→ 6개월 안에 산업안전 + 사고 사례 + 회사 단위 진단 + 자동 맞춤화로 좁은 시장에 깊이 있게 들어가야.
