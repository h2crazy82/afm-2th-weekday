## 3. 따라할 7가지 (Benchmarking To-Do)

### B2. 환급금 자동 계산 + 노출 ★★★

휴넷 표 컬럼: `교육비 / 대기업환급금 / 우선지원환급금`

라라에듀는 단가는 있지만 환급금 노출이 없음. 추가하면:
- 견적 PDF: "고용보험 환급금 — 대기업 ₩X / 우선지원 ₩Y"
- 진단 위저드 결과: "이 패키지의 실제 부담 비용 = 정가 - 환급금"
- 결제 페이지: "환급금 신청 가이드 PDF 동봉"

→ 대기업/중견기업 결제 의사결정자에게 가장 강한 영업 무기.

### B3. 셀프 견적 카탈로그 (다중 선택 → 일괄 견적) ★★

휴넷 스마트견적 흐름:
```
필터 (대/중/소분류 + 난이도 + 환급/비환급) →
과정 검색 → 다중 체크박스 → "선택과정 견적확인" / "검색결과 다운로드"
```

라라에듀에 부족한 것: HRD 담당자가 본인이 원하는 패키지를 직접 조립하는 인터페이스. 현재 PM이 견적 발송을 수동 트리거.

**액션**: paid_content 또는 별도 b2b 사이트에 `/catalog` 라우트 추가:
- 라라에듀 보유 콘텐츠 표 (대/중/소분류 필터)
- 인원 수 입력 → 11구간 단가 자동 계산
- 다중 선택 → "이 패키지 견적 PDF 받기" + "라라에듀 영업팀 연결"
- ENTERPRISE-TARGETING의 /audit과 자연스럽게 연결

### B4. 북러닝 패턴 — 콘텐츠 + 굿즈 결합 ★★

휴넷 "북러닝(도서 제공)" — 도서 + 강의 패키지로 80,000~130,000원. 가격 정당화.

라라에듀에 적용:
- "산업안전 + 안전 점검 체크리스트 PDF + QR 스티커 키트"
- "법정의무교육 + 사내 게시용 포스터 PDF"
- "요즘리더 AI + ChatGPT 프롬프트 모음집 PDF (paid_content와 연결!)"

→ paid_content의 5개 시드 콘텐츠를 위탁교육 패키지에 끼워파는 cross-sell.

### B5. 마이크로러닝 + 몰입형 콘텐츠 카테고리 분리 ★

휴넷 nav 메뉴: `학습솔루션 / 교육콘텐츠 / 마이크로러닝 / 몰입형콘텐츠 / 하이브리드 러닝`

→ 같은 콘텐츠라도 "5분 마이크로러닝" / "VR 몰입" / "오프라인+온라인 하이브리드" 형태별로 분리해 노출. 콘텐츠 재활용 + SEO 우위.

라라에듀 액션: 기존 20차시 요즘리더 AI를 마이크로 1시간/3시간/8시간 버전으로 분리 패키징.

### B6. AI 콘텐츠 빠른 추가 ★★

휴넷이 월 5-10개씩 새 AI 콘텐츠 추가 (제미나이/MAKE/Cursor/AI Agent). 트렌드 캐치업이 빠름.

라라에듀 자산:
- paid_content 프리미엄 기능 (AI가 즉석 콘텐츠 생성) — 이걸 PM이 매주 1개씩 사용해서 새 콘텐츠 자산 자동 축적
- 기존 요즘리더 AI 20차시 + 매주 신규 1개 (52주 / 년) → 1년 후 70개 AI 콘텐츠

### B7. 멤버십 모델 — 휴넷 FLEX 변형 ★

휴넷 FLEX: "법정교육 무료 + 직무교육 무제한 — 중견/중소기업 전용 멤버십"

## 4. 휴넷이 못 하는 5가지 (라라에듀가 깊일 곳)

### D1. 회사별 콘텐츠 자동 맞춤화 ★★★

휴넷은 표준 콘텐츠 카탈로그. "현대미포조선 거제조선소용 화학물질 안전 강의"는 1:1 상담으로만 가능 (느림).

라라에듀 paid_content 프리미엄 기능 (Claude API)이 이 영역에 정확히 위치. **이걸 ENTERPRISE-TARGETING의 진단 위저드 결과 페이지에 통합**하면 자동 맞춤화가 self-serve로.

### D2. 산업안전 + 사고 사례 깊이 ★★★

휴넷은 "산업안전관리"라는 일반 카테고리만. 라라에듀는:
- 산업안전 11구간 단가표 (인원수별 정확한 단가)
- 법정교육 20% 동시할인 로직
- 산업안전보건공단 통계 활용 가능
- 252사 거래처 사고 데이터 (익명 통계화)

→ ENTERPRISE-TARGETING D안 (사고 이력 기반 추천)이 정확히 이 빈자리를 노림.

### D3. 한국 제조업 직무 특화 ★★

휴넷 산업분야별 직무: 금융, 의료/제약, 제조, IT 등 광범위. **제조업만 노리는 깊이는 없음**.

라라에듀: 뚜레쥬르 5개 도급사 (식품 제조업) 운영 경험 + 산업안전 강세 → 한국 제조업 직무에 한정한 전문가 포지션 가능.

### D4. PM 도구 자동화 (paid_content와 연결) ★★

휴넷은 콘텐츠 소비자만 노림. 라라에듀는 콘텐츠 제작자(PM·강사)도 노릴 수 있음:
- paid_content의 AI 프롬프트 마켓 (5개 시드)
- 프리미엄 기능 (즉석 생성)
- 라라에듀 PM 김지원의 효율화 도구 (Q1 MISSION)

→ 콘텐츠 마켓 + 제작 도구 풀스택. 휴넷은 콘텐츠 마켓만.

### D5. 즉시 견적 (자동 산출 + 자동 발송) ★

휴넷 스마트견적도 정가 산출은 가능하지만 **인원수별 협상은 1:1 상담**. 견적 PDF 자동 발송 X.

라라에듀:
- 견적 자동발송 시스템 운영 중 (Python+cron+Google Sheets+Naver SMTP)
- ReportLab 6페이지 PDF 자동
- 5분 진단 → 즉시 견적 PDF 메일 (휴넷보다 1단계 빠름)

→ 이 부분은 이미 강함. 진단 위저드와 합치면 결정타.

## 6. ENTERPRISE-TARGETING.md와의 연결

이 벤치마크는 ENTERPRISE-TARGETING의 5가지 메커니즘을 **휴넷 검증을 통해 강화**:

| ENTERPRISE-TARGETING 메커니즘 | 휴넷 검증 결과 | 강화안 |
|------|------|------|
| A. 회사 진단 위저드 | 휴넷 터닝포인트 큐브 (리더십 개인 진단) — 회사 단위 X | **회사 단위 + 컴플라이언스 + 사고 이력**으로 차별화 강화 |
| B. 법정의무 컴플라이언스 캘린더 | 휴넷 "법정교육 다이렉트" 페이지 별도 운영 | 라라에듀의 20% 동시할인 로직 + 환급금 자동 계산 추가 |
| C. 산업별 표준 커리큘럼 | 휴넷 산업분야별 직무 카테고리만 — 표준 커리큘럼 X | **제조업 한정 깊이 있는 표준 커리큘럼** Phase 2 |
| D. 사고 이력 기반 추천 | 휴넷에는 없음 | **이게 가장 강한 차별화** — 산업안전보건공단 데이터 + 252사 익명 통계 |
| E. 임직원 설문 | 휴넷에는 없음 | Phase 2 보류 |

→ **A + D 조합 추천이 더 강력함이 검증됨**. 휴넷이 정확히 못 하는 영역에 라라에듀의 자산이 정확히 매핑됨.
