# MISSION — 라라에듀 고객사 맞춤 AI 학습 콘텐츠 빌더

> 라라에듀가 이미 보유한 252개 제조업 거래처에게, AI 채팅 한 번으로 "그 회사만의" 직무교육 콘텐츠를 즉석에서 만들어주는 도구.

## 1. 문제 정의

라라에듀는 산업안전·법정의무교육·요즘리더 AI 과정 등을 252개 제조업체에 공급한다. 그런데 매번 같은 콘텐츠를 단순 배포하는 것은 **고객사가 원하지 않는다** — "우리 공장 사례로 바꿔달라", "우리 회사 직급 호칭으로 고쳐달라", "납품기한이 다음 주인데 내일까지 가능?" 같은 요청이 끊이지 않는다.

현장은 이렇게 흘러간다:
- 콘텐츠 제작자(라라에듀 직원 또는 외부 강사)가 **PPT 한 장씩 수동 수정** → 1개 고객사당 4-8시간 소요
- 고객사 HRD 담당자도 **자료를 받기 전까지 무엇이 들어오는지 모름** → 받고 나서 "이거 우리 회사랑 안 맞아요" 재요청
- 그 사이 **20차시짜리 요즘리더 AI 과정** 같은 대형 커리큘럼은 한 번 만들면 5개 고객사에 그대로 복사 — 고객사 입장에선 "다 똑같은 강의" 인식
- 결과: 수주는 가능하지만 **재계약률은 낮음**, 콘텐츠 제작자는 야근, 고객사는 만족도 낮음

핵심은 **"맞춤화의 비용이 너무 비싸다"**는 것이다. AI는 이 비용을 1/100로 만들 수 있다.

## 2. 타겟 유저

**1차 타겟 (도구를 직접 쓰는 사람)**: 라라에듀 콘텐츠 제작 담당자 — 강사 7명 + 내부 PM 2명. 매주 3-5건의 고객사별 맞춤 요청을 받음.

**2차 타겟 (혜택을 받는 사람)**: 제조업체(라라에듀 고객사)의 HRD 담당자 — 평균 40대, 안전관리자 겸직, "내가 받을 자료가 우리 공장 사례에 맞는지" 사전 확인하고 싶음.

**구체 페르소나 — 라라에듀 PM 김지원**:
> 35세, 라라에듀 입사 3년차. 매주 월요일 견적 자동발송 시스템(현재 운영중)으로 신규 문의 4-5건 들어옴. 목요일까지 각 고객사에 맞는 제안서 PDF를 만들어 보내야 함. ReportLab으로 자동 6페이지는 만들지만, 그 안의 "사례" 부분은 매번 수동 — "삼성중공업 거제조선소" "현대자동차 울산공장" 같은 키워드 한 줄 바꾸려고 PPT 30번 켜는 중. 토요일 오전까지 일이 밀리면 가족 약속을 취소함.

## 3. 해결 방법

라라에듀 PM이 챗 한 번으로 고객사 맞춤 콘텐츠를 생성:

```
[고객사 정보 입력]
  - 회사명, 업종, 사업장 위치, 직원 수, 주요 위험 요소
  - 납품 콘텐츠 종류 (산업안전 / 법정의무 / 요즘리더 AI / 커스텀)
  - 차시 수, 분위기 (보수적/혁신적), 호칭 ("팀장님" vs "리더")
       ↓
[AI 채팅으로 1차 초안 생성]
  - Claude API에 라라에듀 콘텐츠 자산 컨텍스트 + 고객사 정보 주입
  - 차시별 학습목표, 핵심 메시지, 우리 공장에 맞춘 사례 3개, 토론 질문
  - 채팅 형태로 "이 부분만 더 디테일하게" 인터랙티브 수정
       ↓
[자료로 정리 → PDF/Markdown 다운로드]
  - 라라에듀 표준 제안서 양식 적용
  - 고객사 로고/CI 색상 자동 반영 (있으면)
  - 차시별 강의 노트 + 평가 문항 5개 자동 생성
       ↓
[고객사별 라이브러리]
  - 같은 고객사의 다음 요청 시 과거 맥락 자동 로드
  - "지난번 산업안전과 톤은 맞추되, 이번엔 AI 윤리로" 가능
```

이게 7주차 MVP의 모습이고, 8주차 발표에는 김지원 PM이 실제로 이 도구로 고객사 한 곳의 1차 초안을 5분 안에 만드는 데모를 보여준다.

## 4. 영어 Tagline (1줄, 5개 후보)

| # | Tagline | 톤 |
|---|---------|---|
| **A** | **AI co-pilot for manufacturing workforce training** | SaaS 표준 — LinkedIn/투자자 톤 |
| B | ChatGPT, but it actually teaches your shop floor | 직설/위트 — 트위터 바이럴 톤 |
| C | Custom factory training, generated in a chat | 결과 강조 — 랜딩 히어로 톤 |
| D | Turn 5-week curriculum design into a 5-minute conversation | 시간 절감 — B2B 영업 메일 톤 |
| E | The AI tutor for Korean manufacturers | 지역+롤 명확 — 국내 보도자료 톤 |

**선정**: 메인은 **A** (SaaS 정체성 명확, LinkedIn HRD 그룹·투자자에게 무난). 랜딩 히어로 카피로는 **C** 병기.

> "**AI co-pilot for manufacturing workforce training** — Custom factory training, generated in a chat."

## 5. 왜 라라에듀가 이걸 만들 수 있는가 (Unfair Advantage)

- **이미 252사 거래처 데이터**가 있음 (CRM Stage1 완료) → 신규 진입자가 만들 수 없는 학습 데이터
- **산업안전·법정의무·요즘리더 AI 콘텐츠 자산** 보유 → AI 학습용 코퍼스
- **견적 자동발송 시스템·뚜레쥬르 5개 도급사 운영** 경험 → B2B 워크플로우 이해도
- **창업자(박훈)가 직접 PM**으로 들어가서 사용하는 dogfooding 가능 → 1번 사용자가 곧 창업자

## 6. Definition of Success (3개월 시점)

- 라라에듀 PM 2명이 매주 5건의 신규 견적 콘텐츠 초안을 이 도구로 생성
- 고객사 맞춤 작업 시간 4-8h → **30분 이하**로 단축 (10배 이상)
- 첫 외부 고객사 1곳에 "이 도구로 만든 콘텐츠"를 공급하고 만족도 인터뷰 1회 진행
- 실패 신호: 김지원 PM이 1주일 써보고 "그냥 PPT가 빠르겠어요" → 즉시 피봇
